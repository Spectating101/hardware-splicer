# Offer Positioning (V1)

## One-Sentence Positioning
Mecha-Splicer gives you repeatable, printable production fixtures so setup is faster and placement is consistent.

## Problem Framing
Current pain:
- Repeating manual alignment wastes time
- Inconsistent placement causes rework
- Existing generic models don't match exact dimensions

V1 promise:
- Generate and print fixture variants quickly
- Standardized build artifacts (`PRINT_PLAN.md`, `PARTS.json`, BOM)
- Clear mechanical constraints and practical limits documented up front

## Packaging
### Product Name
`Fixture Pack V1: Alignment + Repeat Placement + Clamp Base`

### Tiering
- Starter ($19): 1 fixture family + base docs
- Pro ($49): full V1 pack + 3 fixture families + assembly variants
- Studio ($99): Pro + monthly updates for 3 months

## Listing Hooks
- "Stop re-measuring every run"
- "From spec to printable fixture in minutes"
- "Built for desktop printer workflows, not CAD-heavy teams"

## Objection Handling
- "Will this replace engineering simulation?" -> No. It accelerates practical fixture generation.
- "Will this work for my exact setup?" -> Works for defined envelope; custom tolerances still require iteration.
- "Is this production-grade?" -> Prototype-to-light-production fixture grade, not certified industrial tooling.

## Competitive Edge
- Parametric generation + assembly output, not static one-off STL files
- Clear operating envelope and low-friction print workflow
- Can pair with EE workflows for enclosure/mounting expansion in later upsell
