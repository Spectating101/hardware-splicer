# Launch Checklist (V1)

## Product Readiness
- [ ] Core pack generated and zipped
- [ ] `README` with install/use/limits complete
- [ ] `PRINT_PLAN.md` and `PARTS.json` verified
- [ ] `bom.csv` checked for obvious errors

## Proof Assets
- [ ] 6-10 clean screenshots (problem -> fixture -> result)
- [ ] 1 short demo video (60-120s)
- [ ] Before/after workflow example with measured time savings

## Store Setup
- [ ] Gumroad listing published
- [ ] Etsy listing published (if available)
- [ ] Clear refund/support policy posted
- [ ] FAQ for top 5 objections posted

## Pricing + Offer Test
- [ ] Starter / Pro / Studio tiers configured
- [ ] Two listing-copy variants prepared
- [ ] First 14-day KPI targets set

## Operations
- [ ] Support template replies prepared
- [ ] Issue tagging taxonomy ready (fit, print, assembly, expectations)
- [ ] Weekly review slot scheduled for KPI + support feedback

## Launch Done Definition
- [ ] At least 1 paid order
- [ ] First buyer can complete workflow with <=20 min support
