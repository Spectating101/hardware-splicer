# Hardware-Splicer Demo

Frontend authority dashboard for the top-level Hardware-Splicer backend.

```bash
npm install
npm run dev -- --port 5177
```

The seeded project data mirrors backend artifacts generated by:

```bash
python3 scripts/hardware_splicer.py intake --brief examples/intakes/plant_watering_brief.json --out /tmp/hardware_splicer_intake_plant_v2
python3 scripts/hardware_splicer.py intake --brief examples/intakes/rover_brief.json --out /tmp/hardware_splicer_intake_rover_v2
```

The production readiness panel mirrors `PRODUCTION_RELEASE_METRICS.json`: 9 weighted gates covering compile/artifacts, circuit release, mechanical release, actuation release, deterministic simulation, packaging trace, integrated bench, field validation, and reviewed release scope.
