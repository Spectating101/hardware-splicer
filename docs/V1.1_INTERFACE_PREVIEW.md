# v1.1 interface layer — scope reference

**Status:** Shipped in **`v1.1.0`** (stable)  
**Predecessor:** [v1.0.2](../RELEASE_NOTES_v1.0.2.md) bring-up proof

---

## Product category

> **Self-hosted auditable hardware bring-up + design verification workbench**

Not a bring-up checklist alone. Not a browser ECAD suite.

---

## What v1.1.0 ships

| Surface | What it proves |
|---------|----------------|
| **Readiness verdict** | Blockers before fab / power-on |
| **Design verify** | KiCanvas → truth → BOM → fab → exports |
| **Wiring** | Topology Mermaid + narrative guide |
| **Interface lab** | Adapter proving ground (canvas, circuit-json, netlist) |
| **Authority layer** | `PROJECT_PACKAGE`, DRC, bench gates (unchanged moat) |

---

## Out of scope

- Native Windows installer  
- Full tscircuit editor embed  
- Production autoroute as default UX  
- KiCad MCP as hard runtime dependency  
- Public SaaS
